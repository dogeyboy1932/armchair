How it works: The PDF is parsed for text, chunked into topic/description/objective segments, encoded with SciNCL embeddings, and stored in Milvus + PostgreSQL. Similarity scores are computed against all existing courses using a hybrid Dirichlet-JSD + cosine model, then the Neo4j graph is updated. Community detection re-runs automatically.
